수집한 데이터를 기반으로 계산된 각 지표별 평균 효율
데이터입니다.

각 지표별 총 합에서 해당항목이 차지하는 비중을
나타냅니다.

사용자가 특정 광고상품을 보고 해당 광고상품을 클릭한 비율을 나타내는 지표입니다.

네이버 클릭초이스를 본 사용자 수가 1000명이고,해당 광고
상품을 클릭한 숫자가 500일 경우 네이버 클릭초이스의 클릭률은 0.5% 입니다.

* 클릭률(%) =
  클릭수 / 노출수 *100

광고 1클릭당 발생하는 평균비용을 나타내는 지표입니다.

네이버 클릭초이스에 집행한 광고비가 1백만원이고 해당 광고
상품을 클릭한 숫자가 5,000일 경우 네이버 클릭초이스의 클릭비용은 200원 입니다

* 클릭비용(원) =
  광고비 / 클릭수

특정 광고상품을 통해 사이트를 방문한 후 구매, 회원가입 등의 전환을 발생시키는 비율을 나타내는 지표입니다.

네이버 클릭초이스 광고를 클릭한 숫자가 5,000이고 이를 통해
100건의 전환이 일어났을 경우 네이버 클릭초이스의 전환율은 2%입니다.

* 전환율(%) =
  전환수 / 유입수(클릭수) *100

전환 한번 당 발생하는 비용을 나타내는 지표입니다.

네이버 클릭초이스에 집행한 광고비가 1백만원이고 100건의
전환이 일어났을 경우 네이버 클릭초이스의 전환비용은 10,000원입니다.

* 전환비용(원) =
  광고비용 / 전환수

광고비용 대비 매출액을 나타내는 지표입니다.

네이버 클릭초이스에 집행한 광고비가 1백만원이고 이를 통해
1천만원의 매출이 발생했을 경우 네이버 클릭초이스의 ROAS는 1000%입니다.

* ROAS(%) =
  매출액 / 광고비용 *100

광고를 집행한 광고주 수의 비중을 나타내는 지표입니다.

전체 광고주 수가 1000개이고 네이버 클릭초이스를 집행한
광고주 수가 500개일 경우, 네이버 클릭초이스의 집행수(비중)는 50%입니다.

* 집행수 비중(%) =
  해당상품을 집행한 전체 광고주 수 / 전체 광고주 수 *100

특정 광고상품이 노출된 숫자를 나타내는 지표입니다.

사용자들이 특정 광고상품을 클릭한 숫자를 나타내는
지표입니다.

특정 광고상품을 통해 구매, 회원가입, 애플리케이션
다운로드 등으로 전환된 숫자를 나타내는 지표입니다.
